# Студенческое расписание (Indie App)

Мое приложение для управления расписанием пар с поддержкой:
- Автоматического чередования недель (Верхняя/Нижняя).
- Свайп-навигации по дням и неделям.
- Полного сохранения данных в .lesson файл.

### Функционал:
- Добавление пар (Предмет, Тип, Преподаватель, Вещи).
- Поддержка аудиторий и времени.
- Локальное хранилище данных.

### Сборка:
Проект собирается автоматически с помощью GitHub Actions.
